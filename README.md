# AlgoVault

AlgoVault is a personal DSA dashboard built with React that helps developers track their coding journey beyond just solving problems.

Instead of only counting solved problems, AlgoVault stores your patterns, topics, mistakes, notes, favorites, and revision list in one place.

---

## Features

- Goal Tracker
- Progress Dashboard
- Add Problems
- Edit Problems
- Delete Problems
- Favorite Problems
- Revision Tracker
- Search Problems
- Difficulty Filter
- Basic Analytics
- Local Storage Support
